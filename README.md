## Tech Stack
HTML5, CSS3 (Grid), Vanilla JavaScript

## Features
- Set salary and log expenses with live balance calculation
- Data persists via localStorage across browser reloads
- Delete expenses with instant recalculation
- Pie chart visualization using Chart.js
- PDF report download
- Live currency conversion (INR → USD, EUR)
- Threshold alert when balance drops below 10%
- Dark / Light mode toggle
